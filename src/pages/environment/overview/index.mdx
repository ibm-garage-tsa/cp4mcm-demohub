## Cloud Pak for Multicloud Management 

The TechZone Team provides the ability to provision OCP clusters where you can install your own instance of IBM Cloud Pak for Multicloud management.

You can either use the installer available in Catalog in IBM Cloud [here](https://cloud.ibm.com/catalog/content/ibm-cp-management::1-2c621076-b5eb-489d-ae92-c63f13d713ef-global) or use the community installer available in the [GitHub](https://github.com/ibm-garage-tsa/cp4mcm-installer)

Bear in mind that tutorials available on this site were developed for version 2.1, so some details may be change for newer versions.

If you have any issues or feedback, please share it with tutorial authors. Enjoy it!!
